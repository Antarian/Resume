Few examples of my work and some basic theory in my own words. If you want to see them, just browse in the menu or visit About me links.

From technical point of view, this page alone is based on Ionic 4 with Angular 8 and running on AWS (Amazon Web Services).

More precisely it is deployed from GitHub Antarian/Resume codebase through CodeDeploy pipelines to S3 bucket covered with CloudFront with domain setup in Route53.

This website is built as PWA. If you want, you can install this website on your home screen.
